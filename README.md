# swarm-net
This repository contains the IoT firmware, SRAM datasets, and attestation codes used in the paper `Swarm-Net: Firmware Attestation in IoT Swarms using Graph Neural Networks and Volatile Memory` submitted to the IEEE IoT Journal. Our dataset [`IoT Swarm SRAM Dataset for Firmware Attestation`](https://dx.doi.org/10.21227/gmee-vj41) is available on IEEE Dataport.
